CODE

CODE es un sistema de organización institucional que facilita la gestión automática de avisos, tareas, proyectos y exámenes. Ofrece múltiples funcionalidades que contribuyen a una mejor organización y comunicación entre estudiantes, personal docente y administrativo.

Actualmente CODE está en su versión beta funcional 1.0.16

Esta versión incluye las siguientes funcionalidades:

  • Sistema de horarios.
  • Sistema de tareas.
  • Sistema de avisos.
  • Notificaciones, anuncios, tareas y avisos en tiempo real y organizados automáticamente.
  • Subida y descarga de archivos en formato PDF, JPG y PNG.
  • Manejo automático de importación de datos estudiantiles mediante la estructura de archivos MEP.
  • Creación automática de usuarios.
  • Contraseñas cifradas.
  • Tres tipos de usuarios con distintos niveles de acceso.

Fue desarrollado como proyecto para la Expo-Tecnica 2024.

Limpieza Windows

Este script, desarrollado con comandos de PowerShell, está diseñado para facilitar la limpieza y eliminación de programas innecesarios, telemetría y servicios preinstalados y activos en Windows. Su objetivo es reducir la cantidad de recursos y procesos en segundo plano que consume el sistema.

Este script elimina aplicaciones preinstaladas en Windows como Cortana, OneDrive, y otras apps no deseadas. También desactiva servicios innecesarios como la telemetría, la geolocalización y varios servicios relacionados con la administración de hardware y red. Además, desactiva características de privacidad para reducir la recolección de datos y optimizar el rendimiento del sistema.

Es compatible con Windows 10 y Windows 11 y se encuentra en fase de prueba y corrección de errores menores.
